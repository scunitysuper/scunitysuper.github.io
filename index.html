Join the Project
  1. Contact me to add you to this project. Send me your github username only.
  2. Sign into your Github account.
  3. Go to your email account attached to your GitHub account.
  4. Open the invitation email to join this project.
  5. Click the View Invite button in the email.
  6. Click the ACCEPT invite to join this project.

  2. Add an index file to that folder.
  3. Go to this project's repostitory: https://scunitysuper.github.io/
  4. Click on this link to Upload.
  5. (Make sure you are signed in to GitHub... drag the folder with your last name - to the Upload Area
  6. Add a Commit Comment.
  7. Press the Commit button.

  9. visit the folder online, scunitysuper.github.io/lastname where lastnameis the name of the folder you uploaded.
